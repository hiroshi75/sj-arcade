## 🚀 How to Play

```bash
# Clone the repo
git clone https://github.com/hiroshi75/sj-arcade.git
cd sj-arcade

# Start local server
python3 -m http.server 8888

# Open in browser
# http://localhost:8888/
```

## 🛠️ Tech Stack

- Pure HTML5 / CSS3 / JavaScript
- No dependencies
- Retro pixel-art aesthetic
- 8-bit sound effects (Web Audio API)
- CRT scanline effect
